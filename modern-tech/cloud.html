Frequently Asked Questions

Apakah support Docker containers?
Ya, Velos memiliki support native untuk Dockerfile. Cukup sertakan Dockerfile di root repositori Anda, dan kami akan menangani build serta deploymentnya secara otomatis.

Bagaimana dengan data persistence?
Velos adalah platform stateless untuk komputasi. Namun, kami menyediakan integrasi one-click dengan provider database populer seperti MongoDB Atlas, Supabase, dan AWS RDS.

Apakah ada batas bandwidth?
Paket Free memiliki soft limit 100GB. Jika Anda melebihi batas tersebut, kami tidak akan mematikan situs Anda, namun kami akan menghubungi Anda untuk upgrade ke paket Pro.
