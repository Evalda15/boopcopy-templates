Pertanyaan Umum

Bisakah saya mengimport data dari Notion/Evernote?

Ya, kami menyediakan alat import satu-klik untuk Notion, Evernote, dan file Markdown standar. Struktur folder Anda akan tetap terjaga.

Apakah ada aplikasi mobile?

Tentu saja. Aplikasi MonoSpace tersedia untuk iOS dan Android dengan fitur offline-first. Anda bisa bekerja tanpa koneksi internet.

Bagaimana dengan keamanan data saya?

Data Anda dienkripsi saat istirahat (at rest) dan saat transit. Kami menggunakan standar keamanan SOC2 dan backup harian otomatis.
